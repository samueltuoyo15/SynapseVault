# **SynapseVault: AI-Powered Intelligent Storage**

## Overview

SynapseVault introduces DeepKeep AI, an innovative platform providing encrypted, long-term storage designed to think ahead. It intelligently organizes your files with context-aware capabilities, ensuring your data is not just stored, but understood and accessible when you need it most. Experience a new era of smart data management with a modern and responsive user interface built with React and TypeScript.

## Features

-   **Responsive Navigation**: A dynamic navigation bar that seamlessly adapts to various screen sizes, featuring a convenient hamburger menu for mobile devices to enhance user experience.
-   **Context-Aware AI Storage Messaging**: Highlights the core value proposition of intelligent, AI-driven data organization, emphasizing forward-thinking storage solutions.
-   **Prominent Call to Action (CTA)**: Strategically placed "Get Early Access" buttons throughout the landing page to encourage user engagement and conversions.
-   **Modern UI/UX**: Delivers a clean, dark-themed interface with an intuitive layout and engaging visual elements, leveraging custom fonts (`Geist`, `Inter`) and Font Awesome icons for a premium look and feel.
-   **Vite-powered Development**: Benefits from Vite's lightning-fast hot module replacement and efficient build processes, ensuring a highly productive and smooth development experience.

## Usage

To get started with SynapseVault, follow these simple steps to run the application locally or build it for production.

### Local Development

1.  **Install Dependencies**: Navigate to the project directory and install all required packages using npm:

    ```bash
    npm install
    ```

2.  **Start Development Server**: Launch the development server to view the application in your browser:

    ```bash
    npm run dev
    ```

    The application will typically be accessible at `http://localhost:5173` (or another port as indicated by Vite in your terminal).

### Production Build

To create an optimized, production-ready build of the application:

```bash
npm run build
```

This command will compile and minify your code, generating static assets in the `dist` directory, which are ready for deployment to any web server.
